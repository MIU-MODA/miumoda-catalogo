# 🌸 Miu Moda — Catálogo Virtual

Catálogo online de Miu Moda Tienda Virtual Boutique.

## 🔗 Ver catálogo
**[Ver catálogo en línea](https://TUUSUARIO.github.io/miumoda-catalogo)**

## 📦 Actualizar productos
1. Abre **GlowBooks** en tu laptop
2. Ve a la pestaña **Categorías**
3. Haz clic en **⬇️ Descargar mis datos**
4. Renombra el archivo descargado a `catalogo.json`
5. Súbelo aquí en GitHub reemplazando el anterior
6. ¡El catálogo se actualiza en segundos! ✨
